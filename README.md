# linr_backend

## Como rodar

1. Clonar o repositorio

```
git clone https://github.com/Linr-app/linr-backend/
```

2. Inicializar as dependencias

```
yarn
```

3. Rodar:
  * Modo de Desenvolvimento:
  ```
    yarn run start
  ```

  * Modo de Producao:
  ```
    yarn run production
  ```

  * Testes:
  ```
    yarn run test
  ```
